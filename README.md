# Urban Flow Index (UFI)

A graph-based urban traffic congestion analysis project using Python, NetworkX, Tableau, and Streamlit.

---

## 📌 Project Overview

The Urban Flow Index (UFI) is a data-driven framework developed to analyze urban traffic congestion using multiple traffic factors.

The project combines:
- Speed analysis
- Traffic volume
- Road network importance
- Incident severity
- Temporal traffic patterns

to generate a congestion score between 0–100.

---

## 🚦 Features

- Urban Flow Index (UFI) calculation
- Graph-based road network analysis
- Traffic hotspot identification
- Peak-hour congestion analysis
- Incident severity scoring using rule-based NLP
- Interactive data visualization
- Streamlit web application

---

## 🛠️ Tools & Technologies
